
# Vision Agent 项目概览

## 项目简介（Overview）

Vision Agent 是一个基于多模态感知与任务规划的视觉分析智能体系统。

与传统“输入图片 → 输出一句话”的视觉模型不同，本项目将视觉理解抽象为一个 **Agent 过程**：  
Agent 会先根据用户目标生成分析计划（Plan），再逐步调用视觉感知工具（如图像描述、图像问答），最终汇总为结构化结果，并返回完整的执行 Trace，用于调试、复现与评估。

该项目强调 **工程化、可解释性与可扩展性**，而不仅是模型性能。

---

## 项目解决的问题

在真实应用中，单次视觉问答存在以下问题：

- 回答不可控，容易遗漏关键信息
- 无法知道模型“为什么”给出某个答案
- 难以进行多轮交互和调试
- 不具备复现与评估能力

Vision Agent 通过引入 **规划（Planning）、工具调用（Tool Use）和 Trace 机制**，系统性解决了上述问题。

---

## 核心特性

- 🧠 **Agent 规划能力**  
  根据用户目标自动生成分析步骤（Plan）

- 👁️ **多模态感知能力**  
  支持图像 Caption、图像问答（VQA）

- 🔧 **工具化调用**  
  将感知模型作为可调用工具，而非黑盒

- 🧾 **Trace 与可复现性**  
  每次执行自动保存 JSON / Markdown 形式的 Trace

- 💬 **多轮对话与记忆**  
  支持基于 Session 的连续分析

- 📊 **离线评测与反思机制**  
  支持对 Agent 行为策略进行实验与对比

---

## 项目组成模块

Vision Agent 项目由以下核心模块组成：

1. **Perception Service（感知层）**  
   提供图像描述与图像问答能力（test6）

2. **Planning & Trace Engine（规划层）**  
   根据用户目标生成分析计划并执行（test7）

3. **Evaluation & Reflection（评估与反思）**  
   对不同 Agent 策略进行离线评测（test8 / test9）

4. **Web UI（交互层）**  
   提供多轮对话与 Trace 可视化界面

---

## 项目定位

这是一个 **工程型多模态智能体项目**，而非单一模型实验。

项目重点在于：

- Agent 系统设计
- 多模块协同
- 可解释与可评测性
- 从实验到服务的完整路径

---

## 适用场景

- 复杂图像分析任务
- 需要解释与追踪的 AI 系统
- 多轮视觉问答与决策支持
- 多模态 Agent 原型研究
